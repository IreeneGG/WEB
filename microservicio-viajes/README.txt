
Microservicio de Viajes

Descripción:
Este proyecto es un microservicio de viajes desarrollado en Node.js, utilizando Express y MongoDB.

Instalación:
- Asegúrate de tener Node.js y MongoDB instalados en tu sistema.
- Clona este repositorio o descomprime el archivo del proyecto.
- Navega al directorio del proyecto y ejecuta 'npm install' para instalar las dependencias.

Ejecución:
- Para iniciar MongoDB, ejecuta: 'mongod --dbpath data/db'
- Para arrancar el servidor Node.js, ejecuta: 'node main.js'.

Nota: Este proyecto también incluye una configuración de Swagger para la documentación de la API.
